# Archived Code

This directory contains deprecated or archived code that is no longer actively maintained.

## Contents

### deltachat-core/
**Status:** DEPRECATED

The original Delta Chat core library written in C. This has been superseded by [deltachat-core-rust](https://github.com/deltachat/deltachat-core-rust).

The modern deltecho applications use the Rust-based core via JSON-RPC.

**DO NOT use this code for new development.**
